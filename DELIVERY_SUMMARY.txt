CHARACTER SELECTION - 3 MODES:

1. PRESET MODE (Easy)
   ┌─────────────────────────┐
   │ Preset: [Latin ▼]      │
   │ • Latin (A-Z, a-z, 0-9)│
   │ • Devanagari            │
   │ • Arabic                │
   │ • CJK (sample)          │
   └─────────────────────────┘

2. CUSTOM MODE (Flexible)
   ┌─────────────────────────┐
   │ Text: [Hello World   ]  │
   │ → Generates just those  │
   │   8 unique characters   │
   └─────────────────────────┘

3. UNICODE RANGE MODE (Precise)
   ┌─────────────────────────┐
   │ Start: [U+0041       ]  │
   │ End:   [U+005A       ]  │
   │ → Generates 26 chars    │
   │   (A-Z range)           │
   └─────────────────────────┘

UNICODE MAPPING (Why Critical):
┌──────────────────────────────────────────┐
│ Without Mapping:                         │
│ User: types 'A'                         │
│ Font: "Which shape is 'A'?" → Random!   │
│                                          │
│ With Mapping:                            │
│ User: types 'A'                         │
│ Font: "A = U+0041 = Glyph 1" → Correct!│
└──────────────────────────────────────────┘
